Why move to Backdrop?

  • You are on Drupal 7
  • You are happy with your current site
  • You want all the same features you have now
  • You want ongoing updates to your software
  • You want an easy and affordable option

Upgrade from Drupal

in 5 steps

  1. Clean up your Drupal 7 site
  2. Export your database and files from Drupal 7
  3. Set up a Backdrop instance
  4. Import your Drupal 7 site + files to Backdrop
  5. Run the Backdrop update script
